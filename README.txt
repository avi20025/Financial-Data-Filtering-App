Financial Data Filtering App

A React-based application to filter and display financial data with various filters and sorting options

Instructions to run the project locally:

    Prerequisites
        - Node.js installed on your machine
        - Git installed on your machine

    Steps to run:
        1. Clone the repository
            - Open your terminal or command prompt
            - Run the following command to clone the repository:
                git clone https://github.com/avi20025/Financial-Data-Filtering-App.git
        2. Navigate to the project directory
        3. Navigate to the "frontend" directory
        4. Start the development server using the command: npm start
            - Open your browser and go to http://localhost:3000 to view the app.
        5. In a seperate command window, navigate to the "backend" directory. 
        6. Start the backend server using the command: 
            uvicorn main:app --reload
        7. Append the route defined in this app to view the original condensed API data 
            - navigate to this url: http://127.0.0.1:8000/fetch_data
